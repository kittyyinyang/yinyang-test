## 🚀 部署方式（任选一种）

### 方式一：腾讯云 / 阿里云对象存储 COS/OSS（推荐，国内访问快）
1. 登录腾讯云 COS 或阿里云 OSS
2. 创建存储桶，开启静态网站托管
3. 上传全部文件到根目录
4. 获取访问链接，分享到小红书/抖音群

### 方式二：Vercel（国外，免费，简单）
1. 把本文件夹推送到 GitHub
2. 登录 https://vercel.com
3. 导入 GitHub 仓库
4. 自动部署完成，得到一个 `.vercel.app` 链接

### 方式三：GitHub Pages（免费，稳定）
1. 把本文件夹推送到 GitHub 仓库
2. 在仓库 Settings → Pages 中启用
3. 选择 main 分支，保存
4. 几分钟后获得 `xxx.github.io` 链接

### 方式四：直接发给技术人员
把 `yinyang-test-deploy.zip` 文件发给技术人员，让他们上传到你自己的服务器或 CDN。

## 📝 上线前需要修改的地方

在 `index.html` 头部中，把这几个 meta 改成你自己的域名：

```html
<meta property="og:url" content="https://你的域名.com">
<meta property="og:image" content="https://你的域名.com/og-image.png">
```

这样微信群/小红书/抖音分享时才会显示正确的标题和封面图。

## 🎨 自定义内容

- 想换题目：编辑 `index.html` 中 `QUESTIONS` 数组
- 想换结果文案：编辑 `RESULT_DATA` 对象
- 想换角色画像：替换 `images/` 目录下的 4 张 PNG
- 想换配色：修改 CSS 中 `--warm-gold` / `--deep-blue` 等变量

## ⚠️ 注意事项

- 角色画像由 AI 生成，仅用于社群测试，不用于商业用途
- 建议上线前用 3-5 个不同设备测试一遍
- 如果分享到小红书/抖音群，确保域名能在中国正常访问
- 首次测试建议收集 50-100 份数据，再优化题目和结果页
